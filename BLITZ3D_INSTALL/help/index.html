Welcome To Blitz3D SoLoud: MAV-Less Edition!

This is a fork of Blitz3d: SoLoud Edition by BlitzResearch that implements juanjp600's MAV-Less/FastExt Killer/OpenAL patches.
To return to this page at any time, just click the 'home' icon in the toolbar.
