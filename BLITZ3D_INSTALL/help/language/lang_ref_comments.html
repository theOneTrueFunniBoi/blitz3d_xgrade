Comments 

You add comments to your programs using the ';' or "//" characters. Everything following the ';' or "//" until the end of the line will be ignored, this is useful for commenting your code - so you can always look through and follow each line in a logical manner.

The following code shows comments in use;


; Begin the Redraw Function
Function Redraw()
...
End Function

// Begin the Clear Function
Function Clear()
...
End Function

This code also shows a legal use of comments;


Function Redraw() ; Begin the Redraw Function
...
End Function

Function Clear() // Begin the Clear Function
...
End Function

You can also create comments that span multiple lines. The "/*" characters are used to signal the start of a comment, and the "*/" characters the end. Each instance of "/*" or "*/" must appear at the beginning of a line.

The following code shows multi-line comments in use;


; This function is no longer necessary, but will be kept in the source code file just in case.
/*
Function Redraw()
...
End Function
*/

// This function is also no longer necessary.
/*
Function Clear()
...
End Function
*/
