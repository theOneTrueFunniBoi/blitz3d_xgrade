BlitzIdent/BlitzShortIdent/BlitzShortestIdent

Parameters

None.

Description

Returns the current Blitz3D identifier (ex: Blitz3D: MAV-Less).

• BlitzIdent

BlitzIdent returns the full identifier (ex: Blitz3D SoLoud: MAV-Less Edition).

• BlitzShortIdent

BlitzShortIdent returns a shortened version of the Blitz identifier (ex: Blitz3D: MAV-Less).

• BlitzShortestIdent

BlitzShortestIdent returns a very shortened version of the Blitz identifier (ex: Blitz3D).

Example

;BlitzIdent - Example
Text 5,5,"The current Blitz3D identifier is: "+BlitzShortIdent()
